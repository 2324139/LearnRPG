# 修行之路 — S5 Python 古風 RPG 技能樹

> **「問蒼天，何以學道？曰：循序漸進，層層遞升，方能登峰造極」**

一個融合古風敘述、RPG 遊戲化、同儕共學的 Python 教學系統。為香港 S5 ICT 課程量身設計。

## ✨ 特色亮點

- 🎮 **RPG 技能樹系統** — 42 個技能分 6 層級，遞進式解鎖
- 📜 **古風敘述** — 每個技能配古文名稱與古風說明
- 🌳 **可視化進度** — 實時展示學習狀態（鎖定 → 進行中 → 精通）
- 👥 **師徒配對機制** — 5-6 人小班互補共學
- 💾 **Supabase 後端** — 無伺服器架構，數據永久保存
- 🎨 **古風 UI 設計** — 棕金配色、卷軸風格、古籍感十足

## 🚀 快速啟動（3 步）

### 步驟 1: 創建 Supabase 項目（2 分鐘）
```bash
1. 進入 https://supabase.com
2. 點「New project」
3. 設定名稱 & 密碼 & 區域
4. 等待初始化完成
5. 複製 Project URL 與 API Key
```

### 步驟 2: 初始化數據庫（3 分鐘）
```bash
1. 開 Supabase SQL Editor
2. 新建 Query
3. 貼入 supabase-schema.sql 全文
4. 點「Run」執行
5. 確認 6 個表已創建 + 42 個技能已導入
```

### 步驟 3: 配置前端代碼（2 分鐘）
編輯 `skill-tree-ancient.html` 第 300 行：
```javascript
const SUPABASE_URL = 'https://your-project.supabase.co';  // 替換為你的 URL
const SUPABASE_KEY = 'eyJhbGc...';                         // 替換為你的 API Key
```

✅ **完成！** 打開 HTML 文件即可使用

## 📖 使用說明

### 給七兄（教師）
1. **查看班級進度**
   - 在技能樹頁面下拉菜單選學生名字
   - 一覽各生的技能狀態與進度百分比

2. **更新學生進度**
   - 進 Supabase Dashboard → Table Editor
   - 手工編輯 `student_progress` 或用 SQL 批量更新

3. **頒發徽章**
   - 在 `student_achievements` 表添加記錄
   - 系統自動顯示在學生卡上

### 給學生
1. **檢視技能樹**
   - 選自己的名字
   - 查看可用的下一步技能

2. **點擊技能卡**
   - 查看古風說明、難度、預計時長
   - 點擊「保存心得」記錄修習感悟

3. **追蹤進度**
   - 進度條實時更新
   - 顏色變化表示狀態轉換

## ⚠️ 常見問題

**Q: 頁面打開後為空白？**
A: 檢查 API 密鑰配置是否正確。在瀏覽器 F12 → Console 查看錯誤訊息。

**Q: 技能卡顯示但進度為 0%？**
A: 學生進度數據未初始化。在 Supabase Table Editor 手工添加 `student_progress` 記錄。

**Q: 古文顯示亂碼？**
A: 確認 HTML 字符編碼為 UTF-8。在瀏覽器右鍵 → 字符編碼 → UTF-8。

## 📞 技術支援

若遇問題，請檢查：
1. Supabase 項目是否創建 & SQL 腳本是否執行
2. HTML 中的 URL & API Key 是否正確替換
3. 瀏覽器 Console (F12) 是否有紅色錯誤
4. 學生進度數據是否已初始化

## 📝 設計理念

> **古風 RPG 框架的教學意義**

傳統 RPG 遊戲的五大要素：
- 🎮 **角色成長** → 學生進度升級，從「初入山門」到「大道已成」
- 🗺️ **迷宮探索** → 技能樹就是知識迷宮，需循序而行
- ⚔️ **挑戰戰役** → 月度挑戰賽模擬「武林大會」
- 🤝 **同盟協作** → 師徒配對與同儕共學
- 💎 **寶藏獎勵** → 徽章、排行、特殊稱號激勵內驅力

古風敘述不只是裝飾，而是**認知框架的轉變**：
- 從「被動上課」到「主動修行」
- 從「分數競爭」到「技能精進」
- 從「個人奮鬥」到「宗派傳承」
